## Sistema de Gerenciamento de Eventos 

Projeto final desenvolvido para a disciplina de Projeto e Modelagem de Banco de Dados.
Conta com um diagrama relacional que exemplifica a relação das entidades, atributos e dados do banco.
(Diagrama Relacional.png)

O projeto utiliza postgresql para criar o banco e suas tabelas, assim como JDBC para integrar o banco de dados com uma interface do terminal, usando o método PreparedStatement() para maior segurança
O Projeto é passivel de aprimoramentos.
